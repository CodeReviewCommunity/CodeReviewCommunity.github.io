Code Review Community

This community is dedicated to the Weekend Challenges and other projects going on on Code Review.

Open an issue on this repo to get added to the organization. Please state what language/team(s) you would like to be added to. If your reputation on Code Review is greater than or equal to 2000, you will be added to the "Admin" team automatically.
